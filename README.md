# M7 - Acceso a datos en aplicaciones Node

Este proyecto tiene como objetivo construir la conexión entre el servidor Express y una base de datos relacional.

## Prerrequisitos o Dependencias

Lista de software y herramientas, incluyendo versiones, que necesitas para instalar y ejecutar este proyecto:

- Node.js
- Express
- PostgreSQL

## Instalación del Proyecto

Una guía paso a paso sobre cómo configurar el entorno de desarrollo e instalar todas las dependencias.

```bash
# paso 1: clonar el proyecto
git clone https://github.com/koerdev/M7_Acceso_datos_aplicaciones_node.git
```

```bash
# paso 2: Entrar al proyecto
cd proyecto
```

```bash
# paso 3: instalar las dependencias
npm install
```

```bash
# paso 4: crear archivo .env con la variable del puerto
PORT = 3000
```

## Instrucciones para Ejecutar el Proyecto

Instrucciones para ejecutar el proyecto una vez instalado.

```bash
# ejecutar el proyecto en local
npm run dev
```

```bash
# prueba de rutas
https://localhost:3000/
https://localhost:3000/status
https://localhost:3000/logs
https://localhost:3000/log
```
